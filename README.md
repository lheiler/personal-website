# Lorenz Heiler - Personal Portfolio

A personal portfolio website bridging Mechanistic Neuroscience, Scalable Systems, and Creative Direction.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Configured for Cloudflare Pages

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure
- `app/`: Main application routes and layout.
- `components/ui/`: Reusable UI components (BentoGrid, etc.).
- `components/sections/`: Content sections (Hero, Research, Engineering, CreativeLab).

## Deployment
This project includes a `wrangler.toml` configuration for deployment on Cloudflare Pages.
